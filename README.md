# Zonkai-app
Plataforma completa de Gestão Empresarial construída com foco em eficiência, estética moderna (Glassmorphism e Dark Mode) e experiência do usuário (UX/UI fluida).  Este repositório contém o código Frontend da aplicação ZonKai, incluindo a estrutura HTML e os estilos CSS complexos que implementam o tema escuro dinâmico e a navegação responsiva.
